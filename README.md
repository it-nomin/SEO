# SEO ツール

初心者がSEOに役立つツール。複数URLから SEO関連情報を一括取得できます。

## 機能

- **title 一括取得** - ページの `<title>` タグを抽出
- **meta description 一括取得** - メタディスクリプション属性を抽出
- **canonical タグ取得** - 正規化URL（canonical）を抽出
- **複数URL処理** - テキストエリアに改行区切りでURL入力可能

## 使用方法

1. ブラウザで `index.php` を開く
2. テキストエリアにURLを入力（改行区切り）
3. 「Check!」ボタンをクリック
4. HTML テーブルで結果を表示

## 取得情報

| 項目 | 説明 |
|------|------|
| URL | 入力したURL（クリック可能なリンク） |
| title | ページのタイトル |
| meta description | メタディスクリプション |
| canonical | 正規化URL |

## セキュリティ対応

- ✅ **XSS対策** - すべてのHTML出力を `htmlspecialchars()` でエスケープ
- ✅ **タイムアウト対応** - URLアクセス時に5秒のタイムアウト設定
- ✅ **エラーハンドリング** - 失敗時は「[取得失敗]」などの安全なメッセージを表示

## 技術仕様

- **言語**: PHP 3
- **文字コード**: UTF-8（自動判別対応）
- **日本語対応**: mbstring 関数で多言語対応
- **外部ライブラリ**: なし

## 関数一覧

### `getPageTitle($url)`
URLからページタイトルを取得します。

**戻り値**: 文字列（失敗時は「[取得失敗]」または「[title未検出]」）

### `getMetaDescription($url)`
URLからmeta description を取得します。

**戻り値**: 文字列（未検出時は「[未検出]」）

### `getCanonical($url)`
URLから canonical タグを取得します。

**戻り値**: 文字列（検出されない場合は「[検出されず]」）

## 対応フォーマット

- **URL形式**: `https://example.com`
- **複数URL入力**: 改行区切り（CRLF/LF自動判別）
